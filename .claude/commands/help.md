## Claude Code Commands

### 📋 Task Master (`/project:tm/`)
Complete task management system with natural language understanding.

**Get Task Master help:**
```
/project:tm/help
```

**Quick examples:**
- `/project:tm/init` - Initialize project
- `/project:tm/list` - List tasks
- `/project:tm/next` - Get next task
- `/project:tm/workflows/smart-flow` - Smart workflows

### 🚀 Getting Started

1. **New to Task Master?**
   ```
   /project:tm/setup/quick-install
   /project:tm/learn start
   ```

2. **Already installed?**
   ```
   /project:tm/help
   /project:tm/status
   ```
